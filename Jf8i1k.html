문제

아래 설명된 의미에 해당하는 용어를 쓰세요 
 반복문으로 for( A; B; C ){ D }로 구성되며, 초기화 A나 증감식 C를 수행하고, 조건식 B를 확인 후 참이면 명령문 D를 반복적으로 수행하고 거짓이면 반복문 중단하고 빠져나옴. 참고로, D는 0번 이상 실행  

 비교연산자로 왼쪽 값이 오른쪽 값보다 크면 참을 반환하고 아니면 거짓을 반환  

 감소연산자로 연산자의 왼쪽 변수 공간에 저장된 값에 오른쪽 값만큼 감소시켜 왼쪽 변수 공간에 저장  

 복합대입연산자로 연산자의 왼쪽 변수 공간에 저장된 값에 오른쪽 값만큼 더한 값을 왼쪽 변수 공간에 저장
