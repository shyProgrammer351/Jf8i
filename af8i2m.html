문제

반지름을 입력받아 원둘레와 원면적을 구하는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
반지름을 입력하세요 : 3
반지름 3.00인 원둘레는 18.85이고 원면적은 28.27입니다.
반지름 ( 3.00 + 1.00 )인 원둘레는 25.13이고 원면적은 50.27입니다.

프로그램 코드

	#include <stdio.h> 

	#define PI 3.141592				
1	#define CIRCUMFERENCE( r )  ( 2 * PI * ( r ) )		
2	#define CIRCULARAREA( r ) ( PI * ( r ) * ( r ) ) 		
	
3	int main()																													
	{
		double radius = 0.0;						
		double circumference = 0.0;	
4		double circularArea = 0.0;	

		printf("반지름을 입력하세요 : ");																				
5		scanf("%lf", &radius );																						

6		circumference = 
7			 	CIRCUMFERENCE( radius );																			
8		circularArea = 
9			 	CIRCULARAREA( radius );																				

10		printf("반지름 %.2f인 원둘레는 %.2f이고 원면적은 %.2f입니다.", 							
						radius, circumference, circularArea );										

11		circumference = 
12			 	CIRCUMFERENCE( radius + 1 ); 																
13		circularArea = 
14			 	CIRCULARAREA( radius + 1 );																	

15		printf("반지름 ( %.2f + 1.00 )인 원둘레는 %.2f이고 원면적은 %.2f입니다.", 		
						radius, circumference, circularArea);											
	
16		return 0;
	}
